fork 操作：就是从别的大佬远程端 git仓库 拷贝一份到自己的 git仓库。

在开发学习时，经常通过 fork 别人的 git 开源项目进行开发学习。
所以在我们开发完成后，想合并到大佬的主分支去，我们需要进行创建一个 merge request， 提交给主分支代码管理员进行代码审核，审核通过后允许 merge，这样我们开发的代码就会合并到主项目中。
